Police Patrol Route Optimization

This is an exploration of an application of Euler's graph rules to find possible route loops for patrol vehicles to follow in order to cover all roads in a certain area while minimizing the amount of roads covered multiple times. The methods studied also serve to find snowplow routes and any other variation of the Chinese Postman Problem, especially those that involve weighting different roads in order to promote greater coverage of such roads.

This repository supplements a paper written as part of a combinatorics class.

Overview

  • Parses portion of GeoJSON file as provided by local county GIS department into GraphML file.
  • Attempt to Eulerize GraphML file, which turns the graph (using the fewest possible changes) into one containing an Euler Circuit by adding duplicate edges, representing duplicated roads being covered in one patrol cycle.
  • Finds Euler circuit in Eulerized graph using either NetworkX module's built-in functions or a variation of dijkstra's algorithm as taught in class (Keller & Trotter 2017) and self-implemented.
  • Unparses Euler Circuit from graphml into GeoJSON
  • Visualizes GeoJSON file live & saves visualized file

Technologies Used
