ReferAI

ReferAI is a web application that performs comprehensive symptom questioning, provides preliminary diagnostics, and recommends nearby doctors best suited to serve the user's needs.

Features

  • Performs a ~2 minute symptom questioning in the form of a chatbot.
  • Provides preliminary diagnoses.
  • Generates anonymous PDF report of questioning for use by doctor/nurse.
  • Recommends doctor based on doctor rating & user location.
  • Allows users to create accounts in order to remember location, age information.

Technologies Used

  • Python
  • Streamlit
  • SQLite
  • Google Gemini
  • HTML/CSS
