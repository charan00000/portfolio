SpendAI

SpendAI is a web application that lets users create budgets, moniter spending, and recieve financial advice.

Features

  • Allows users to create budgets and view spending amounts based on different spending categories (food, transportation, etc).
  • Allows users to vew spending over time.
  • Allows users to input expense transactions and input income.
  • Features a chatbot which offers financial advice and uses user's financial profile as context.
  • Allows users to create accounts in order to remember inputted budgets & transactions.

Technologies Used

  • Javascript
  • React
  • Python
  • Django, Django DB
  • HTML/CSS
