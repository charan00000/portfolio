GTMovieStore

GT Movie Store is a web application that lets users search through a catalog of movies and "buy" movies.

Features

  • Allows users to scroll through a catalog or search for a movie name.
  • Allows users to add movies to cart.
  • Allows users to "purchase" movies in cart, generating a confirmation code and saving the transaction in 'My Orders' page.
  • Allows users to write reviews and ratings for each movie.
  • Allows users to create accounts in order to remember orders and reviews.
  • Allows administrators to update movie catalog through a seperate, password-protected admin page.
  • Allows administrators to delete reviews, add/delete orders for different users.

Technologies Used

  • Python
  • Django, Django DB
  • HTML/CSS
