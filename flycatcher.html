FlyCatcher

FlyCatcher is a small GameBoy Advanced video game programmed in C that lets users move a net around and catch moving insects.

Features

  • Allows users to keep score of how many flies they have catched. Score decrements when a fly moves across the screen without being caught.
  • 'Win' screen loads when score exceeds 5.
  • 'Lose' screen loads when scores goes below 0.
  • Generates random names for each fly. Upon losing, the user is taunted with the names of the first five flies to escape (make it across the screen without being caught).

Technologies Used

  • C
  • DMA Utilization
