HistoreX

HistoreX is a web application that generates a short educational video based on an inputted historical event.

Features

  • Can read user-inputted textbook chapter and parse text into context.
  • Generates script for video using user-inputted historical event and (optional) textbook context.
  • Generates speech recording of script.
  • Generates relavent images to be displayed as frames as a slideshow, with script context used to generate images relevant to associated portion of script.
  • Combines script and images into mp4 video to be displayed

Technologies Used

  • Python
  • Streamlit
  • Google Gemini (script generation) & Vertex (image & sound generation)
  • OpenCV
